About

The web interface of MetaboAnalyst was implemented based on the open-source JavaServer Faces Technology using the PrimeFaces library (v12.0). The back-end computations are based on the open-source MetaboAnalystR package. Please note the raw spectra processing module has been developed as an independent R package (optiLCMS) for the ease of installation and maintenance.

To support growing user traffic, MetaboAnalyst 5.0 has employed distributed computing and microservice for scalable metabolomics data analysis. The new spectral processing module also allows user registration in order to perform this time-consuming task. Due to its complex configurations and sensitive information stored in the package, the support for local installation of the web server has been discontinued. We encourage users to use the public web servers for flexible and interactive data analysis tasks, and the R packages for batch processing.
